## steps of changes
- Make Changes
- add files
`git add .`
- Push code 
`git push`



## To change the master to main branch 
- Changing the repo
`git branch -M main`

- Adding the Remote Git With orgin 
`git remote add origin URL`

- push Your all code on Git Repo
`git push -u origin main`

- to publish your branch 

`git push --set-upstream origin branchname`

- Local Branch to set upstream 

- code push directly when git push

`git push -u origin feature-branch`

`git push`


- to up-to-date local branch with the Remote Branch

`git pull`
